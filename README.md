# Régie Vidéo - Panneau Tactile

Interface tactile pour régie vidéo : gestion caméras PTZ, médias (génériques/synthés), conducteur d'émission.

## Démo

Ouvrir `index.html` dans un navigateur ou déployer via GitHub Pages.

## Fonctionnalités

- Moniteur PGM (Programme) avec sélection caméras
- Preview caméra avec zoom et auto-framing
- Grille caméras 3×3 + D-pad de contrôle PTZ
- Conducteur d'émission
- Gestion des médias (génériques DIRECT, MORNING, etc.)
- Détection de visages
- Filtres par catégorie et intervenant
